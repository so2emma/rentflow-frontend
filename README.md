# RentFlow Frontend

This folder contains the React application built using **Next.js (App Router)**, **Tailwind CSS**, and **TypeScript** for the RentFlow platform.

## 🛠 Technology Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v3 (Utility classes configured via `tailwind.config.js`)
- **Language**: TypeScript
- **State Management**: React Context or Zustand
- **Routing**: Next.js File-system Routing

## 📁 Source Layout
- `src/app`: Application pages and layout routes (Layout, Home, Login, SignUp, Admin, Landlord, Tenant).
- `src/components`: Reusable UI components (e.g., [ProtectedRoute](file:///Users/eoso/IdeaProjects/RentFlow/frontend/src/components/ProtectedRoute.tsx)).
- `src/services`: API clients and interceptors (e.g., [api.ts](file:///Users/eoso/IdeaProjects/RentFlow/frontend/src/services/api.ts)).

Please refer to the **[README.md](file:///Users/eoso/IdeaProjects/RentFlow/build_phases/README.md)** in the `build_phases` directory for detailed incremental build specifications.
